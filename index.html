• HOUSING & DEVELOPMENT BOARD RESALES
  • About this
    • Goal
    (1) User can research on HDB resales price between 2017 to Feb 2020.
    (2) Filter by town, street-name, year, room, storey, area & model down to your specific needs.
    (3) Return the results in table form. Show on map the location of each result.

    Guide
    (1) Start by filtering by town or street names. If no options are selected, default will be the street name displayed on dropdown i.e. ADMIRALTY DR.
    (2) Optional filtering by year, room, storey, area & model. If no options are selected, default will include all year, room, storey, area & model.
    (3) Hide filters to remove filters from view.
    (4) Generate results in table by ascending or descending price order. Table limit display to 50 results. Continue click "show/next" buttons for next 50 results.
    (5) The last column in the table can be checked to show the location of result on the map. Otherwise, the table's header last column can be checked to show all 50 results at once on map.
    (6) The map top-right allows user to search location by inserting address.
    (7) Press 'Reset' button to clear tables and markers from map. Previous selections will not be resetted.


    Source
    Data courtesy from data.gov.sg.
